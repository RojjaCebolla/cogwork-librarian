# [cogwork-librarian](https://coglib.sosk.watch/)

## prereqs

- node. i manage my node installations with [asdf](https://github.com/asdf-vm/asdf)

## local dev setup

```shell
git clone git+https://github.com/dsoskey/cogwork-librarian.git
cd cogwork-librarian
npm install
npm run dev-server
```

## tests to run
```shell
npm run test
```
